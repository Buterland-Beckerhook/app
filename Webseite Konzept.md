# Webseite-Konzept: Schützenverein Buterland-Beckerhook e.V.

> Navigationskonzept für das Redesign. Jede Hauptkategorie bekommt ein
> Sub-Menü. Historische Inhalte aus der AGENTS.md (beim letzten Redesign
> verloren gegangen) werden wieder integriert.

## Hauptnavigation (Top-Level)

```
Aktuelles | Termine | Verein | Thron | Kontakt
```

---

## „Der Verein" — Sub-Menü

Bisher 5+1 Seiten, wird auf 10+ ausgebaut.

```
Über uns
├── Vereinsgeschichte         ← Vorgeschichte, Gründung 1909, Hofbrand 1911
├── Vereinsfahne & Insignien  ← Fahnenhistorie (1924/1961/1967), Königsketten,
│                               Thronbild (Erich Zgodda, 1990)
├── Vereinslokal              ← Jägerhaus (1908–2004), Dinkelhof (seit 2004)
├── Ehrenmal                  ← Errichtet 1934, renoviert 1997, 20 Gedenktafeln,
│                               Kranzniederlegung am Schützenfestsonntag
├── Schützenfest              ← Ablauf Sa–Di, Festplatz, Programm
├── Letztes Jubiläum          ← Festschrift, Bildergalerie, historische Rückblicke
│   (z.B. 110 Jahre, 115 Jahre)

Vorstand
├── Aktuelle Vorstandsmitglieder + Aufgaben
├── Präsidenten (seit 1909)   ← Biografien aller Amtsinhaber (Eimann, Dropmann,
│                               Lewing, Gleis-Preister, Verst, Frieske)
├── Kontakt zum Vorstand

Offiziere
├── Aktuelle Offiziere + Aufgaben, Dienstgrade
├── Chefs der Offiziere       ← Biografien aller Oberste seit 1909 (Sundag,
│                               Lammers, Tönjann, Hoff, Terhalle, Droste)
├── Kontakt zu den Offizieren

Jungschützen ("Die Besten im Westen")
├── Geschichte                ← Gründung 1956, Holzgewehre, erste 14 Mitglieder
├── Jubiläum                  ← Jungschützenbogen 1984/2016, Bildergalerie
├── Die Jungschützen heute    ← ~45 Mitglieder, Anzugsordnung,
│                               Aufnahme/Biertaufe, Grünholen
├── eigene Königswürde        ← Jungschützenschießen (seit 1966), eigene Kette

Ausschüsse                    ← NEU: Sammelseite
├── Kinderfestausschuss (KFA) ← eigenständiger Ausschuss, Kinderfest am
│                               Schützenfest-Samstag
├── Seniorenausschuss         ← gegründet 1998, Ausflüge (Meyer Werft,
│                               Hengstparade, Feuerwache …)
├── Schießriege               ← gegründet 1988, Pokale (Schützen-, Junioren-,
│                               Senioren-, Königs-, Biwak-Pokal)

Mitglied werden
├── Aufnahmeantrag (PDF)
├── Beitragsordnung / Beitragsfreiheit
├── Voraussetzungen (16 Jahre, Wohnsitz, 3 Jahre für Königsschuss)

Satzung                       ← NEU: Volltext der Satzung vom 03. Dezember 2005
├── Kernpunkte (Vereinsbezirk, Mitgliedschaft, Organe, Wahlen)

Ehrenmitglieder               ← NEU: Liste aller bekannten Ehrenmitglieder
├── (Nacke, Gründken, Niehoff, Hoff, Dornhege, Kühlkamp, Hork, Duesmann,
│   Blömer, Thebelt u.a.)

Nachbarvereine                ← NEU: befreundete Vereine aus Gronau/Epe
├── Stadtkapelle Gronau, Spielmannszug KAB Epe
```

## „Thron" — Sub-Menü

Bislang nur eine Seite; wird mit Unterseiten angereichert.

```
Aktueller Thron 2025/2026
├── Königspaar, Ehrenpaare, Oberhofmarschall, Mundschenk

Alle Throne seit 1909         ← komplette chronologische Liste

Kaiserthron 1984
Kaiserthron 2009

Königstreffen                 ← NEU: alle 5 Jahre, Fotos
Königinnentreffen             ← NEU: alle 5 Jahre, Fotos
```

---

## „Aktuelles" — bleibt, mit Jahres-Archiv

```
Jahrgangs-Archiv (jährlich, seit 2019)
Kategorien / Tags
```

## „Termine" — bleibt, mit iCal-Abo

## Optionale Ergänzungen (Screens)

| Bereich | Inhalt |
|---|---|
| **Schießergebnisse** | Jährliche Ergebnisse Pokal- und Preisschießen, Biwak-Pokal, Stadtmeisterschaften, Vergleichsschießen |
| **Bildergalerien** | Pro Event eine Galerie (Schützenfest, Biwak, Winterfest, Frauenfest, Kuhfladenbingo, Weihnachtsmarkt) |
| **Download-Bereich** | Aufnahmeantrag, Satzung (PDF), Festzeitschriften (1984, 2009 …) |
| **Glossar** | Vogelschießen, Königsschuss, Biwak, Kehraus, Grünholen, Vogelbauer, Stadtschützenfest, Euregio-Schützenfest, Königs-/Königinnentreffen |

## Zusammenfassung der Änderungen

| Aspekt | Alt (live) | Neu (Konzept) |
|---|---|---|
| Sub-Menüs | Nur bei /Verein (flach) | Alle Kategorien mit Sub-Navigation |
| /Verein | 5 Seiten | 10+ Seiten |
| Historie | kaum sichtbar | Vertieft: Präsidenten, Oberste, Fahne, Ehrenmal, Vereinslokal |
| Ausschüsse | nur KFA erwähnt | KFA, Senioren, Schießriege eigenständig |
| Satzung | fehlt | Volltext |
| Ehrenmitglieder | fehlt | Eigene Seite |
| Schießergebnisse | fehlt | optional |
| Thron | flache Liste | Mit Unterseiten + Treffen-Historie |
